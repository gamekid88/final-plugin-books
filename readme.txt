== Description ==

Creates a list of books with authors, summary, and thoughts about the book in a list format.Allows the admin to list all books or
books by a certain author. Also has a global settings page that will allow the admin to display a different title. 

= Make Suggestions Or Contribute =
Book Logging System is on [GitHub](https://github.com/gamekid88/final-plugin-books)!


== Installation ==


* Navigate to Add New Plugin page within your WordPress
* Search for Book Logging system
* Click Install Now link on the plugin and follow the prompts
* Activate the plugin through the 'Plugins' menu in WordPress

**Or**

* Upload plugin dir to the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
